## â“ FAQ

**Q: Do viewers need to create accounts?**  
A: No! Public gallery requires zero login.

**Q: Where are images stored?**  
A: In browser localStorage (your device).

**Q: Will images disappear?**  
A: Only if you clear browser cache. They persist otherwise.

**Q: Can I limit who downloads?**  
A: No, anyone with the link can download.

**Q: How many images can I upload?**  
A: Limited only by browser storage (~50-100).

**Q: What image formats work?**  
A: PNG, JPG, WebP (max 5MB each).

**Q: Do I need a server?**  
A: No! Everything works client-side.

**Q: Can I password protect?**  
A: Not with this version (would need backend).

**Q: Do images sync across devices?**  
A: No, each device is separate. Login in each device.

**Q: Can I delete images?**  
A: Yes, from dashboard. They disappear from gallery too.
